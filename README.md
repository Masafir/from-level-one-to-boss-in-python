# 🐍 From Level One to Boss in Python

> Programme d'apprentissage intensif Python — De développeur React/Node.js à **Fullstack Python Boss** en 6 mois.

---

## 🎯 Objectif

Maîtriser Python au niveau professionnel pour postuler à des postes **Fullstack Python** :
- API REST / GraphQL avec **FastAPI**
- Data engineering : **Pandas**, **ETL pipelines**
- Event-driven architecture : **asyncio**, **Celery**, **Redis**
- WebSockets, message queues, CQRS
- Le tout combiné avec **React** en frontend
